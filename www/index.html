supraHex: An open-source package for tabular omics data analysis using a supra-hexagonal map

The package supraHex is developed to meet the need for quickly understanding genome-wide biological data, which usually involve a large number of genomic coordinates (e.g. genes) but a much smaller number of samples. It devises a novel map shape called supra-hexagonal map (a giant hexagon on a 2-dimensional (2D) map grid seamlessly consisting of smaller hexagons) to self-organise the input data. This trained map is then used for integrated tasks, highlighted by simultaneous analysis of genes and samples, and the overlaying of additional data (if any) onto the map for multilayer omics data comparisons.

2 Availability

Available at http://r-forge.r-project.org/projects/suprahex as part of R-Forge, and also available as part of Bioconductor (see stable release version and latest development version).

Tips for the installation:

  • stable release version 
    
> source("http://bioconductor.org/biocLite.R")
> biocLite("supraHex")
  • latest development version (highly recommended). 
    
> install.packages("hexbin",repos="http://www.stats.bris.ac.uk/R")
> install.packages("supraHex",repos="http://R-Forge.R-project.org", type="source")

    More instructions on how to install are detailed in http://supfam.org/supraHex.

    • 4 Realcase

    To showcase the multifaceted functionalities accomplished by supraHex, we used published multilayer omics datasets on DNA replication timing, promoter CpG classification and gene expression (see Figure 1). These datasets consist of digitised replication timing, promoter CpG status and expression levels of 17,292 genes in a variety of samples. The DNA replication timing data matrix is first used as the input for training, producing a replication timing map (see Figure 2). Then, additional CpG and expression data are overlaid onto the trained replication-timing map, revealing an inherent relationship between late replication, low CpG density promoters and low expression levels (see Figure 3). Full descriptions can be found in supraHex publication.

    Figure 3. Using the replication-timing map to intuitively explore relationships between replication timing, CpG and expression.
    (A) CpG data overlaid onto the replication-timing map. The colour bar indicates the probability of containing genes with HCPs (left panel) or LCPs (right panel); the stronger, the higher.
    (B) Expression data overlaid onto the replication-timing map. Each map illustrates a sample-specific expression profile. Here genes with similar replication-timing patterns are still mapped onto the same position on the map, but the illustrations are replaced with expression patterns..
